What even is FART IDE?

FART IDE is an experimental, node-based development environment designed for anything you can can dream of. Build games, apps, interactive experiences, and more — all without writing boring code.

If you like node graphs, strange abstractions, and software that feels alive instead of corporate, you’re in the right place. Note: FART IDE is still in early development, so expect bugs. A LOT of them.
